# my-farm-integrated-dashboard

Generate a self-contained, single-page Observable Plot HTML dashboard for corn soil analysis.

## When to invoke

- Building an integrated agricultural dashboard
- Analyzing soil quality across multiple fields for corn production
- Generating a downloadable, offline HTML report
- Comparing field performance with NDVI-validated soil quality metrics

## Starting point

Run `scripts/generate_dashboard.py` to produce the HTML artifact.

See `README.md` for full documentation.
