# Strategy Index

Use this area for agronomic planning, crop-risk framing, and maturity decision support.

- [Crop Strategy Guide](crop-strategy/GUIDE.md) - crop planning references and report-ready recommendations for corn, soybean, cotton, wheat, and sorghum.
- [Maturity by FIPS Guide](maturity-by-fips/GUIDE.md) - county/FIPS maturity workflows for shared corn RM and soybean MG assets.

Use crop strategy when the request asks what to plant, how to frame a 2026 crop plan, or how to translate field data into grower recommendations. Use maturity-by-FIPS when the request asks for county-scale maturity lookup outputs, annual maturity map builds, or shared maturity tables.
